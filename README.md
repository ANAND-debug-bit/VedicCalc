<h3>Why I made this website ?</h3>

<hr>

<p>
  I made this website because I want to make that typical calculations way more faster through the methods of Vedic Maths which was given by Jagadguru Shankaracharya Swami Bharati Krishna Tirthaji . He developed this system of mental calculation techniques between 1911 and 1918 . So to spread awareness about this fantastic mathematical innovation I built this website with a Vedic period themed website .
</p>

<h2>Some cool features</h2>

<ul>
<li> Home page with a classical Vedic mathematics theme with a rotating mandala art icon animation on the home page. </li>
<li> 4 algebra operation cards on the homepage as well as on the navigation bar.</li>
<li> Each algebra operation has 4 learning cards with both explanation and examples.</li>
<li> Practice Test section also available to boost your confidence level.</li>
<li> 5 Levels , starting from Level 1 (beginner) upto level 5 (expert).</li>
<li> Computer based Test format view. </li>
<li> With custom time setter in each level as per your confidence level. </li>
<li> After marking each question answer -- detailed solution also there giving reference to the learning card in that particular algebra operation. </li>
<li> Result view has No. of correct and incorrect questions as well as percentage of acuuracy with retry option also available. </li>
<li> A motivation statement after each result view based on the accuracy percentage. </li>
  <li> Core Vedic Sutras at the bottom of the site.</li>
</ul>
